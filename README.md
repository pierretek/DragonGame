# Dragon Game &nbsp; <img src="https://github.com/user-attachments/assets/b5cfbcb1-679e-4902-8f90-763de67edfeb" width="40" height="40" align="top" />

A simple text-based game about fighting a dragon in a boss fight

## Details 
- Main class is in `src/main/Main.java`
- The entire game is within a single java file
- Each turn the dragon can either heal itself, or attack you
- After that, you can pick from six possible actions to respond
- Reduce the dragon’s health to zero before it does the same to you!

## Requirements 
- Java JDK version 19 or higher
- *Netbeans IDE (if you wanna build it yourself)*

## How to Run
1. Download `DragonGame-Playable.zip` from [releases](https://github.com/pierretek/JFramery/releases/tag/playable)
2. Extract it and open the folder in a terminal
3. Run ``java -jar DragonGame.jar`` to run the jar file
4. Enjoy!

*If you want to build this project yourself, you can clone this repo and import it into Netbeans as a project.*

## Actions
Each round you can one of six possible actions:
- **1 - Attack**
  - Directly attack and damage the dragon with your sword
- **2 - Heal**
  - Take a turn to recover some of your health
- **3 - Run away and scream**
  - Run away and abandon the fight like a coward
- **4 - Drink mystery potion**
  - Drink a mystery potion that has a 50% chance of either doubling or halving your next attack
- **5 - Roll magic dice**
  - Roll a magic dice that deals 6,000 damage to the dragon if it lands on a six
